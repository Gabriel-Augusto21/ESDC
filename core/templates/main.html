Software para Cálculo e Gestão da Dieta de Equinos:
Tecnologia a Serviço da Nutrição e Bem-Estar Animal

A nutrição equina é essencial para a saúde, desempenho e bem-estar dos cavalos, mas muitos criadores enfrentam dificuldades na formulação de dietas equilibradas devido à falta de conhecimento técnico, à complexidade dos cálculos nutricionais e ao difícil acesso a ferramentas adequadas. Embora existam diretrizes confiáveis para alimentação equina, sua aplicação prática ainda é limitada, especialmente entre pequenos criadores. Diante disso, este projeto de extensão propõe o desenvolvimento de um software acessível e intuitivo para cálculo e gestão de dietas equinas, com base em parâmetros científicos, visando facilitar a elaboração de planos alimentares personalizados conforme as necessidades específicas de cada animal.

A nutrição equina é essencial para a saúde e desempenho dos cavalos, mas muitos criadores enfrentam dificuldades por falta de conhecimento técnico e acesso a ferramentas adequadas. Este projeto propõe um software intuitivo e acessível, baseado em parâmetros científicos, para facilitar a formulação de dietas personalizadas para diferentes categorias de equinos.

Gerenciamento de Alimentos

  • Pesquisa dos alimentos

    • O sistema possui ampla base de dados com diversos alimentos que podem compor a dieta dos animais. Você pode pesquisar facilmente o alimento que deseja dentro dessa variedade e encontrá-lo com praticidade.

  • Adição, remoção e atualização de dados alimentares

    Um sistema de nutrição eficiente exige flexibilidade. Por isso, desenvolvemos uma interface intuitiva para que você possa cadastrar, editar ou remover alimentos, nutrientes, e composições alimentares, garantindo precisão e controle a qualquer momento.

Gerenciamento de Exigências

  • Atualização de dados de exigências

    • Nosso sistema permite o cadastro e a edição das exigências nutricionais de acordo com a categoria do cavalo. De forma simples e organizada, é possível manter atualizados os valores exigidos para cada nutriente, facilitando o controle e a gestão dos dados.

  • Pesquisa de exigências

    • Encontre rapidamente as exigências nutricionais já cadastradas no sistema por meio de uma busca eficiente. Você poderá buscar por categoria do cavalo, ou seja, o nome da exigência, facilitando a visualização e o gerenciamento das informações disponíveis.

Gerenciamento de Animais

  • Cadastro e organização

    • O sistema permite registrar todos os animais de forma clara e organizada, incluindo nome, idade, categoria, peso e demais informações essenciais. Assim, você mantém um controle completo dos equinos e acessa rapidamente os dados de cada um.

  • Pesquisa e atualização dos dados

    Com a busca integrada, é possível localizar qualquer animal com facilidade. Além disso, os dados podem ser atualizados a qualquer momento, garantindo precisão no acompanhamento nutricional e na gestão individual de cada cavalo.

Gerenciamento de Dietas

  • Cadastro e balanceamento da dieta

    • O sistema oferece ferramentas para montar dietas personalizadas de acordo com as necessidades nutricionais do animal, fazendo o cálculo automático do balanceamento da dieta, possibilitando criar a melhor dieta possível. É possível adicionar alimentos e visualizar a contriubuição de cada um na nutrição do animal de forma simples e rápida.

  • Visualização e edição

    • Visualize as dietas nutricionais já cadastradas no sistema por meio de uma listagem eficiente das mesmas. O sistema também oferece a possibilidade de editar as dietas cadastradas quando quiser, facilitando o controle da alimentação do animal.
